# Employee Management System (EMS)
A simple (in-memory) Employee Management System built using `HTML`, `CSS`, and `TypeScript`.

### How to run:
- clone/download project repo and `cd` into it
- run the following commands:
  - `npm install`
  - `npx tsc`
  - `npm start`
